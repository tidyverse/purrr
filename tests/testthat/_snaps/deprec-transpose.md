    Code
      . <- transpose(list())
    Condition
      Warning:
      `transpose()` was deprecated in purrr 1.0.0.

    Code
      transpose(1:3)
    Condition
      Error in `transpose()`:

# elements of input must be atomic vectors

    Code
      transpose(list(environment()))
    Condition
      Error in `transpose()`:
      ! Element 1 must be a vector, not an environment.

    Code
      transpose(list(expression(a)))
    Condition
      Error in `transpose()`:
